# UrgenceGabon.com - Guide des fonctionnalités

Ce document présente tout ce que vous pouvez faire avec la plateforme, que vous soyez un simple visiteur, un pharmacien, un annonceur ou un administrateur.

---

## Pour les visiteurs

### Trouver une pharmacie

Dès l'arrivée sur le site, vous voyez la liste des pharmacies. Vous pouvez :

- **Chercher par nom** : tapez "Pharmacie du Soleil" dans la barre de recherche
- **Filtrer par ville** : cliquez sur "Libreville", "Port-Gentil" ou une autre ville
- **Voir les pharmacies de garde** : un onglet dédié montre celles ouvertes 24h/24
- **Utiliser la carte** : repérez les pharmacies autour de vous et cliquez pour plus de détails

### Consulter les détails

Pour chaque pharmacie, vous trouvez :

- Nom et adresse
- Numéros de téléphone (cliquez pour appeler directement)
- Horaires d'ouverture
- Services proposés (parapharmacie, matériel médical, conseil...)
- Position sur la carte

### Accéder aux urgences

L'onglet "Numéros d'urgence" liste :

- Les numéros nationaux (police, pompiers, SAMU)
- Les numéros locaux par ville (hôpitaux, cliniques, protection civile)

Chaque numéro est cliquable pour appeler immédiatement.

### Contribuer

Vous connaissez une pharmacie qui n'est pas listée ? Les horaires affichés sont faux ? Vous pouvez aider :

- **Soumettre une localisation GPS** : indiquez les coordonnées exactes d'une pharmacie
- **Corriger une information** : proposez un nouveau numéro de téléphone ou de nouveaux horaires
- **Suggérer une pharmacie** : décrivez une pharmacie manquante pour qu'on l'ajoute
- **Envoyer une suggestion** : partagez vos idées pour améliorer le site

Toutes les contributions sont vérifiées avant publication.

## Pour les pharmacies

### Être référencé

Votre pharmacie peut apparaître sur la plateforme avec :

- Nom et coordonnées complètes
- Horaires précis
- Liste des services
- Position exacte sur la carte
- Badge "vérifié" après validation

### Être en pharmacie de garde

Quand vous êtes de garde, votre pharmacie est mise en avant :

- Affichage prioritaire dans les résultats
- Icône distinctive sur la carte
- Présence dans l'onglet "Pharmacies de garde"

### Mettre à jour vos informations

Si quelque chose change (téléphone, horaires, services), signalez-le via le formulaire de contact ou demandez à un utilisateur de soumettre la correction.

## Pour les annonceurs

### Créer une publicité

Le système publicitaire permet d'afficher :

- **Images** : uploadez votre visuel (JPG, PNG, etc.)
- **Vidéos** : intégrez une vidéo YouTube, Facebook ou autre

Chaque publicité inclut :

- Un titre accrocheur
- Un texte descriptif
- Un bouton d'action ("En savoir plus", "Profiter de l'offre"...)
- Un lien vers votre site ou page

### Configurer l'affichage

Vous contrôlez quand et comment votre pub apparaît :

- **Dates de diffusion** : programmez le début et la fin de votre campagne
- **Priorité** : les pubs à haute priorité sont montrées plus souvent
- **Délai "Passer"** : temps avant que l'utilisateur puisse fermer la pub (5 secondes par défaut)

### Suivre les performances

Pour chaque publicité, vous voyez :

- Nombre de vues (combien de fois elle a été affichée)
- Nombre de clics (combien de personnes ont cliqué)
- Taux de clic (pourcentage de conversion)

## Pour les administrateurs

### Tableau de bord

L'interface d'administration donne une vue d'ensemble :

- Nombre total de pharmacies
- Pharmacies de garde actives
- Soumissions en attente de validation
- Graphiques de fréquentation (7 jours, 30 jours)
- Top des pharmacies les plus consultées

### Gestion des pharmacies

Vous pouvez :

- **Ajouter** une nouvelle pharmacie avec toutes ses informations
- **Modifier** une pharmacie existante
- **Supprimer** une pharmacie obsolète
- **Activer/désactiver la garde** avec dates de début et fin
- **Valider la position GPS** soumise par un utilisateur

### Traitement des contributions

Quand un visiteur envoie une correction ou une suggestion :

1. Vous la voyez dans la liste des soumissions en attente
2. Vous vérifiez l'information
3. Vous approuvez ou refusez
4. Si approuvé, la modification est appliquée automatiquement

### Contacts d'urgence

Vous gérez la liste des numéros d'urgence :

- Ajout de nouveaux services (hôpitaux, cliniques...)
- Modification des numéros existants
- Organisation par ville ou niveau national
- Ordre d'affichage personnalisable

### Paramètres du site

Vous personnalisez l'apparence :

- Nom du site
- Logo et favicon
- Image de partage sur les réseaux sociaux
- Description pour le référencement
- Code de suivi analytics

### Popups d'information

Vous créez des messages qui s'affichent aux visiteurs :

- Annonces importantes
- Messages de bienvenue
- Avertissements
- Avec ou sans image

### Configuration publicitaire

Vous définissez les règles globales :

- Activation/désactivation des pubs
- Déclencheur : temps, nombre de pages, ou au rechargement
- Limite par session
- Délai entre deux pubs
- Affichage mobile et/ou desktop
